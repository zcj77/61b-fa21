Flood Puzzle

The goal of this puzzle is make all the cells have the same color. The cell in the upper-left corner and all cells that are reachable by a series of horizontal and vertical steps that touch only cells of the same color is called the active region. Clicking on a cell of any color differing from that of the active region changes all cells in the active region to that new color.

Each puzzle has a limit on the number of cells that may be selected. The puzzle must be completed within this number of moves.
